데이터에 자유를! 범용 분석 대시보드 (Smart Insight)
매번 새로운 데이터를 볼 때마다 코드를 새로 짜거나 엑셀 필터를 거는 게 번거로웠습니다.

그래서 "어떤 CSV를 던져도 파이썬이 알아서 요리해주는 도구"가 있으면 어떨까? 하는 생각으로 시작한 프로젝트입니다.

💡 이 프로젝트를 통해 해결하고 싶었던 것
분석가로서 데이터를 다룰 때 가장 귀찮은 건 '반복되는 전처리'와 '뻔한 시각화'입니다.

"나이는 평균을 내야 하고, 보험료는 합산해야 하는데..."

"매번 컬럼명 확인해서 코드 고치기 너무 귀찮은데?"

이런 고민들을 자동화 로직으로 풀어냈습니다. 단순히 숫자를 띄우는 게 아니라, 데이터의 헤더를 읽어 분석 메뉴를 스스로 제안하는 똑똑한 대시보드를 지향합니다.

🔥 제가 공들여 만든 핵심 포인트들
1. "눈치껏" 작동하는 분석 엔진
파이썬(Pandas)이 컬럼명을 슥 훑어보고 성격을 파악합니다.

age가 보이면? -> 알아서 '세' 단위를 붙이고 평균을 계산합니다.

premium이 보이면? -> '₩' 기호를 붙이고 합계를 냅니다.

사람이 일일이 지정해주지 않아도 데이터에 맞는 옷을 입혀주는 과정에 가장 공을 들였습니다.

2. 살아있는 동적 버튼 UI
데이터를 올리는 순간 상단에 버튼들이 생깁니다.

이건 고정된 버튼이 아니라, 데이터의 컬럼에 따라 생성되는 맞춤형 메뉴입니다. 버튼을 누를 때마다 차트가 슥슥 변하는 경험을 통해 데이터 탐색이 즐거워지도록 설계했습니다.

3. 실무적인 트러블슈팅
개발 과정에서 컬럼명 끝에 붙은 공백 하나 때문에 에러가 나는 상황을 겪었습니다.

사용자는 완벽한 데이터를 올리지 않을 수도 있다는 생각에, 어떤 데이터가 들어와도 튕겨내지 않도록 strip() 처리와 예외 로직을 꼼꼼하게 배치했습니다.

🛠 사용한 기술들
Frontend: Next.js, Tailwind CSS (애플 스타일의 깔끔한 UI를 위해 선택했습니다.)

Backend: Python (FastAPI), Pandas (데이터 분석의 근본 조합이죠.)

Visualization: Recharts (웹에서도 부드러운 차트 애니메이션을 보여주고 싶었습니다.)

📈 앞으로의 계획
지금은 기본적인 통계와 시각화 위주지만, 앞으로는 이 대시보드 하단에 Tableau 상세 분석 페이지를 연결해서 '빠른 요약'과 '심층 분석'을 동시에 할 수 있는 구조로 확장할 예정입니다.
